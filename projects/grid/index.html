Simple lightweight Grid layout

INCLUDE:
<script src="js/gridlayout.js"><script>
USAGE:
var grid = new GridLayout('promos-wrapper', {
   selector: '.promo',
   speed: 400,
   colwidth: 200,
   debuglog: false,
   animate: true
});
OPTIONS:
selector: '.promo', CSS selector for grid elements animate: false, Include css animation when position changes centerwrapper: true, Center wrapper element
speed: 500, Speed of css animation in miliseconds (works when option animate is set to true) easing: 'linear', Animations easing, CSS format colwidth: 400, Set colwidth for easer computing
resizedelay: 200, Delay on resize debuglog: false, Write in console basic informations debugelems: false Write in console geometry properties for all grid elements

free for use and modification
